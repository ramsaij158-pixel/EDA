# Exploratory Data Analysis Practice

An EDA practice notebook using the Seaborn `tips` dataset to understand distributions, outliers and relationships between variables.

## Project Goal

The goal of this project is to practice visual data analysis and learn how charts help explain patterns in a dataset.

## Dataset

The notebook uses the public Seaborn `tips` dataset, which includes restaurant bill, tip, day, time and customer-size information.

## Analysis Covered

- Histogram of total bills
- Frequency analysis using bins
- Boxplot by day
- Outlier detection using IQR
- Scatter plot of total bill vs tip
- Interactive Plotly visualization
